## Testing Different Perspectives

### As Super Admin
- Login with: `superadmin@schoolmgmt.com`
- Can manage all tenants
- Can create new schools
- System-wide analytics

### As School Admin
- Login with: `admin@demo.com`
- Can manage all school data
- Create/edit staff, students
- Configure school settings

### As Principal/Vice Principal
- Login with: `principal@demo.com` or `viceprincipal@demo.com`
- View all academic data
- Monitor attendance, performance
- Generate reports

### As Teacher
- Login with: `teacher@demo.com`
- View assigned classes
- Mark attendance
- Enter grades
- Create timetable

### As Student
- Login with: `student@demo.com`
- View own profile
- Check timetable
- View grades
- Access assignments

### As Parent
- Login with: `parent@demo.com`
- View child's profile
- Check attendance
- View grades
- Communicate with teachers

### As Librarian
- Login with: `librarian@demo.com`
- Manage books
- Issue/return books
- Track fines

### As Transport Incharge
- Login with: `transport@demo.com`
- Manage routes
- Track vehicles
- Student allocations
